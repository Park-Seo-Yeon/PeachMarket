## 🚀기대 효과

<ul>
<li> 보장성 : 옷 핏 및 자신의 피부색과의 어울림을 가상 피팅을 통해 미리 확인함으로써 구매 실패를 방지
<li>편의성 : 판매자와 구매자간의 사이즈와 관련한 반복적인 문답 및 치수 정보 획득을 위한 인터넷 조사의 번거로움 감소
</ul>
